## 🎯 Як тестувати

### Крок 1: Підготовка

1. Імпортувати всі файли в TIA Portal
2. Створити інстанс DB:
```
   DB_Test_Routes_Instance : FB_Test_Routes
```
3. Додати виклик в OB (Cyclic Interrupt 100ms або Main)

### Крок 2: Запуск сценарію

**Online mode в TIA Portal:**

1. Відкрити `DB_Test_Routes_Config`
2. Встановити:
```
   TestEnabled = TRUE
   RouteId = 1
   TestScenario = 1  (простий START→DONE)
```
3. Спостерігати за:
   - `DB_PlcToScada_RouteStatus.RoutesSts[1].RS_State`
   - `TransitionCount` (має збільшуватись при змінах стану)
   - `Snapshot_*` (останній перехід стану)

### Крок 3: Аналіз результатів

**Очікуваний результат для Сценарію 1:**

| Час | State | ResultCode | ActiveStep | Примітка |
|-----|-------|------------|------------|----------|
| 0s | IDLE (0) | 0 | 0 | Початок |
| 0.5s | VALIDATING (1) | 0 | 0 | Після START |
| 0.5s | STARTING (2) | 0 | 0 | Lock OK |
| 0.5s | RUNNING (3) | 0 | 0 | Виконання Step 0 |
| ~3-4s | RUNNING (3) | 0 | 1 | Після RUNNING редлера 0 |
| ~6-7s | RUNNING (3) | 0 | 2 | Після RUNNING редлера 1 |
| ~6-7s | DONE (6) | 200 | 2 | Всі кроки виконані |
| ~9s | IDLE (0) | 0 | 0 | Після RT_CMD_NONE |
